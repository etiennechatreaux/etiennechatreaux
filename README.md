# About Me

I’m a Data Scientist but not just writing Jupyter notebooks.
Many of my repositories are private, but here’s an overview of the projects I actively maintain or experiment with.
Feel free to reach out if you’d like to discuss my work.

Currently working on:<br>
🎓 **Master Thesis — Non-linear Dimensionality Reduction in Finance**: Exploring how Kolmogorov–Arnold Networks (KANs) can rethink classical factor models.<br>
📊 **Quantitative Research**: Developing data-driven strategies in sports betting and concert ticket arbitrage.<br>
📱 **Growable — Personal Growth App**: Building mobile & web platform that delivers structured feedback and personalized recommendations.<br>

## 🗂️ Old projects

### 🧠 Machine Learning & Data Science

- **DimRed-KAN-Finance** — Master thesis using Kolmogorov–Arnold Networks as nonlinear autoencoders for dimensionality reduction on S&P 500 returns.  
- **Credit_Project_ML** — Complete credit scoring pipeline with preprocessing, modelling and evaluation.  
- **GAN_generator** — Experiments on GAN-based synthetic data generation with realism benchmarking.  
- **Diabete_Projet** — Predictive modelling on diabetes dataset, feature exploration and classification.  
- **WaitingTimePred** — Predictive modelling for queueing / wait-time estimation.  
- **ML-from-scratch** — Linear regression and basic models implemented from scratch, experimenting with custom hyperparameters and training logic.

---

### 📈 Quantitative Finance

- **BlackScholesCppSimulator** — C++/Python simulator of Black–Scholes with analytical vs Monte Carlo pricing for multiple payoffs.  
- **gpu_mc_pricer** — Python Monte Carlo pricer designed for GPU/CUDA acceleration and Rust backends.  
- **rust-montecarlo-pricer** — Fully Rust-based Monte Carlo pricing engine with a focus on speed and safety (API coming soon).  
- **Stresstest_Generation** — Generator of stress scenarios for portfolio risk analysis.

---

### 🧩 Algorithmic Trading & Automation

- **crypto_bot** — Backtesting tools and algorithmic crypto strategies.  
- **projet_arbitrage** — Statistical arbitrage with performance analysis and experiments.  
- **tickets-miam-bot / tickets-acc-manager** — Bots automating queues, triggers, and session management for ticketing platforms.  
- **dofus3_bot / grepo_bot** — Game bots automating movements, resource collection, and UI interactions.  
- **automatisations-n8n** — n8n workflows for API integrations, notifications, and automated data sync.
